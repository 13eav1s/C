# Тесты для лабораторной работы NoLL
     ## Входные данные
     Целые xa, ya, xb, yb, xc, yc.
     ## Выходные данные
     С плавающей точкой P
     ## Позитивные тесты:
     - 01 - обычный тест;
     - 03 - все числа минусовые;
     - 04 - все числа 0;
     ...
     - 08 - есть число 200 и 100.
     ## Негативные тесты:
     - 01 - Вместо цифр буква;
     - 02 - Вместо последнего числа идёт буква;
     ...
     - 06 - Вместо цифр знаки.
